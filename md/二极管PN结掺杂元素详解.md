# 二极管 PN 结掺杂元素详解

**整理时间**: 2026-03-30  
**信息来源**: 半导体物理教材、集成电路工艺资料

## 一、核心结论

| 区域 | 掺杂类型 | 掺杂元素 | 价电子数 | 多数载流子 |
|------|---------|---------|---------|-----------|
| **P 区** | P 型半导体 | **硼(B)**、铝(Al)、镓(Ga) | 3 价 | 空穴（正电荷） |
| **N 区** | N 型半导体 | **磷(P)**、砷(As)、锑(Sb) | 5 价 | 电子（负电荷） |

**最常用组合**：
- **P 区**：硅 + **硼(B)**
- **N 区**：硅 + **磷(P)** 或 **砷(As)**

## 二、半导体基础

### 2.1 基体材料

| 材料 | 符号 | 价电子数 | 应用 |
|------|------|---------|------|
| **硅** | Si | 4 价 | 最常用（95%+ 芯片） |
| **锗** | Ge | 4 价 | 早期器件，现较少用 |
| **砷化镓** | GaAs | 化合物 | 高频、光电器件 |

### 2.2 掺杂原理

**本征半导体（纯净硅）**：
```
硅原子：4 个价电子
晶体结构：每个硅原子与周围 4 个硅原子形成共价键
导电性：很差（载流子浓度低）
```

**掺杂后**：
| 掺杂类型 | 掺杂元素 | 价电子 | 效果 |
|---------|---------|--------|------|
| P 型掺杂 | 硼(B) | 3 个 | 缺少 1 个电子 → 产生**空穴** |
| N 型掺杂 | 磷(P) | 5 个 | 多余 1 个电子 → 产生**自由电子** |

## 三、P 型半导体（Positive）

### 3.1 掺杂元素

| 元素 | 符号 | 原子序数 | 价电子 | 使用频率 |
|------|------|---------|--------|---------|
| **硼** | B | 5 | 3 | ⭐⭐⭐⭐⭐ 最常用 |
| 铝 | Al | 13 | 3 | ⭐⭐⭐ |
| 镓 | Ga | 31 | 3 | ⭐⭐ |
| 铟 | In | 49 | 3 | ⭐ |

### 3.2 硼掺杂详解

**为什么硼最常用？**

| 因素 | 说明 |
|------|------|
| 原子半径 | 与硅接近，晶格畸变小 |
| 固溶度 | 在硅中固溶度高，可达 10²⁰/cm³ |
| 扩散系数 | 适中，工艺可控 |
| 成本 | 相对较低 |
| 工艺成熟 | 应用历史长，工艺稳定 |

**掺杂反应**：
```
硅晶格中：Si - Si - Si - Si
掺入硼后：Si - B - Si - Si
           ↑
       缺少 1 个电子 → 形成空穴
```

**能级位置**：
- 硼在硅中形成**受主能级**
- 受主能级位置：价带上方约 0.045 eV
- 室温下即可电离，产生空穴

### 3.3 P 型半导体特性

| 特性 | 说明 |
|------|------|
| 多数载流子 | 空穴（带正电） |
| 少数载流子 | 电子 |
| 费米能级 | 靠近价带 |
| 导电机制 | 空穴导电为主 |
| 电阻率 | 可通过掺杂浓度调节 |

## 四、N 型半导体（Negative）

### 4.1 掺杂元素

| 元素 | 符号 | 原子序数 | 价电子 | 使用频率 |
|------|------|---------|--------|---------|
| **磷** | P | 15 | 5 | ⭐⭐⭐⭐⭐ 最常用 |
| **砷** | As | 33 | 5 | ⭐⭐⭐⭐⭐ 常用 |
| 锑 | Sb | 51 | 5 | ⭐⭐⭐ |

### 4.2 磷 vs 砷

| 对比 | 磷(P) | 砷(As) |
|------|-------|--------|
| 原子半径 | 较小 | 较大 |
| 扩散系数 | 较大（扩散快） | 较小（扩散慢） |
| 固溶度 | 约 10²¹/cm³ | 约 10²¹/cm³ |
| 能级位置 | 导带下方 0.044 eV | 导带下方 0.049 eV |
| 主要应用 | 一般掺杂、阱区 | 源漏区、浅结 |

**为什么砷也常用？**
- 扩散系数小 → 适合**浅结**工艺
- 激活率高 → 电阻率低
- 适合**源漏区**重掺杂

### 4.3 N 型半导体特性

| 特性 | 说明 |
|------|------|
| 多数载流子 | 电子（带负电） |
| 少数载流子 | 空穴 |
| 费米能级 | 靠近导带 |
| 导电机制 | 电子导电为主 |
| 电子迁移率 | 高于空穴迁移率（约 2-3 倍） |

## 五、PN 结形成

### 5.1 结构示意

```
        P 区                    N 区
    (掺硼硅)                (掺磷硅)
    空穴多                  电子多
       ↓                      ↓
  +  +  +  +  |  -  -  -  -
  +  +  +  +  |  -  -  -  -
  +  +  +  +  |  -  -  -  -
       ↑                      ↑
   多数载流子：空穴        多数载流子：电子
               
               ↑
          耗尽层/空间电荷区
```

### 5.2 耗尽层形成

| 过程 | 说明 |
|------|------|
| 扩散 | P 区空穴向 N 区扩散，N 区电子向 P 区扩散 |
| 复合 | 扩散的载流子在界面附近复合 |
| 空间电荷 | P 区留下负离子，N 区留下正离子 |
| 内建电场 | 从 N 区指向 P 区，阻止进一步扩散 |
| 平衡 | 扩散电流 = 漂移电流 |

## 六、掺杂工艺

### 6.1 主要工艺方法

| 工艺 | 原理 | 适用场景 | 温度 |
|------|------|---------|------|
| **扩散法** | 高温下杂质原子扩散进入硅片 | 深结、阱区 | 900-1200°C |
| **离子注入** | 高能离子束轰击硅片 | 精确掺杂、浅结 | 室温 + 退火 |
| **外延掺杂** | 生长掺杂外延层 | 特殊结构 | 600-1100°C |

### 6.2 离子注入 vs 扩散

| 对比 | 离子注入 | 扩散法 |
|------|---------|--------|
| 精度 | 高（可精确控制剂量和深度） | 较低 |
| 温度 | 低温（后续需退火激活） | 高温 |
| 均匀性 | 好 | 一般 |
| 成本 | 高 | 低 |
| 应用 | 现代 IC 主流工艺 | 传统工艺、深结 |

### 6.3 典型掺杂浓度

| 区域 | 掺杂类型 | 典型浓度 (cm⁻³) | 电阻率 |
|------|---------|----------------|--------|
| 衬底 | P 型或 N 型 | 10¹⁴-10¹⁶ | 1-100 Ω·cm |
| 阱区 | N 阱/P 阱 | 10¹⁶-10¹⁷ | 0.1-1 Ω·cm |
| 源漏 | N+/P+ | 10¹⁹-10²¹ | 0.001-0.01 Ω·cm |
| 接触区 | N++/P++ | >10²⁰ | <0.001 Ω·cm |

## 八、掺杂浓度计算

### 8.1 载流子浓度

**N 型半导体**：
```
电子浓度：n ≈ ND (施主浓度)
空穴浓度：p = ni²/ND
```

**P 型半导体**：
```
空穴浓度：p ≈ NA (受主浓度)
电子浓度：n = ni²/NA
```

其中：
- ni = 本征载流子浓度（硅，300K 时约 1.5×10¹⁰/cm³）
- ND = 施主浓度（N 型掺杂）
- NA = 受主浓度（P 型掺杂）

### 8.2 电阻率计算

**N 型**：
```
ρ = 1/(q × n × μn)
```

**P 型**：
```
ρ = 1/(q × p × μp)
```

其中：
- q = 电子电荷 (1.6×10⁻¹⁹ C)
- μn = 电子迁移率（硅中约 1350 cm²/V·s）
- μp = 空穴迁移率（硅中约 480 cm²/V·s）

## 九、常见问题

### 9.1 为什么硅是最常用的半导体材料？

| 因素 | 说明 |
|------|------|
| 储量 | 硅是地壳中第二丰富的元素 |
| 成本 | 原材料成本低 |
| 氧化层 | SiO2 是优质绝缘层 |
| 工艺 | 工艺成熟，设备完善 |
| 温度 | 工作温度范围宽 |

### 9.2 为什么 N 型迁移率高于 P 型？

- 电子迁移率（μn）≈ 1350 cm²/V·s
- 空穴迁移率（μp）≈ 480 cm²/V·s
- **电子迁移率约为空穴的 2.8 倍**

**原因**：
- 电子有效质量小于空穴有效质量
- 电子散射概率低于空穴

**影响**：
- N 型器件速度更快
- CMOS 中 PMOS 尺寸通常比 NMOS 大 2-3 倍

### 9.3 掺杂浓度对器件性能的影响

| 参数 | 高掺杂 | 低掺杂 |
|------|--------|--------|
| 电阻率 | 低 | 高 |
| 击穿电压 | 低 | 高 |
| 结电容 | 大 | 小 |
| 开关速度 | 快 | 慢 |
| 漏电流 | 大 | 小 |

## 十、总结

### 10.1 核心知识点

| 问题 | 答案 |
|------|------|
| P 型掺什么元素？ | **硼(B)** 最常用，铝(Al)、镓(Ga)也可用 |
| N 型掺什么元素？ | **磷(P)** 或 **砷(As)** 最常用，锑(Sb)也可用 |
| 为什么选这些元素？ | 价电子数不同（3 价或 5 价），原子半径与硅接近 |
| 多数载流子是什么？ | P 型：空穴；N 型：电子 |
| 掺杂浓度范围？ | 10¹⁴-10²¹/cm³（根据应用需求） |

### 10.2 记忆口诀

```
P 型掺硼三价元，空穴导电带正电
N 型掺磷五价元，电子导电带负电
PN 结处耗尽层，单向导电是特性
```
