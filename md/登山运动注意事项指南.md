## 一、登山前准备

### 1.1 身体准备

#### 体能训练（提前 4-8 周）

| 训练项目 | 频率 | 说明 |
|---------|------|------|
| 有氧运动 | 3-4 次/周 | 跑步、游泳、骑行，每次 30-60 分钟 |
| 腿部力量 | 2-3 次/周 | 深蹲、箭步蹲、台阶训练 |
| 核心力量 | 2-3 次/周 | 平板支撑、仰卧起坐 |
| 耐力训练 | 1-2 次/周 | 长距离徒步、爬楼梯 |

#### 健康检查

| 检查项目 | 必要性 | 说明 |
|---------|--------|------|
| 心肺功能 | ⭐⭐⭐⭐⭐ | 心电图、肺功能测试 |
| 血压检查 | ⭐⭐⭐⭐⭐ | 高血压患者需谨慎 |
| 关节检查 | ⭐⭐⭐⭐ | 膝、踝关节状况 |
| 血常规 | ⭐⭐⭐ | 贫血者不宜高海拔登山 |

**不宜登山人群**：
- ❌ 严重心脏病患者
- ❌ 严重高血压患者（血压>160/100）
- ❌ 严重哮喘患者
- ❌ 近期手术恢复期
- ❌ 孕妇（尤其孕早期和晚期）
- ❌ 严重贫血患者

### 1.2 路线规划

#### 路线选择原则

| 原则 | 说明 |
|------|------|
| 量力而行 | 根据体能和经验选择难度 |
| 循序渐进 | 从低海拔、短路线开始 |
| 了解路况 | 提前查询路线难度、长度、爬升 |
| 预留余量 | 计划时间预留 20-30% 余量 |
| 备选方案 | 准备下撤路线和应急预案 |

#### 难度等级参考

| 等级 | 适合人群 | 海拔 | 时长 | 爬升 |
|------|---------|------|------|------|
| 休闲级 | 新手、家庭 | <1000m | 2-4 小时 | <300m |
| 入门级 | 有基础者 | 1000-2000m | 4-6 小时 | 300-600m |
| 进阶级 | 有经验者 | 2000-3500m | 6-8 小时 | 600-1000m |
| 挑战级 | 专业登山者 | 3500-5000m | 8-12 小时 | >1000m |
| 专业级 | 专业登山者 | >5000m | 多日 | - |

## 二、登山过程中的注意事项

### 2.1 行进技巧

#### 上坡技巧

| 要点 | 说明 |
|------|------|
| 小步慢走 | 步幅小、频率稳，避免大喘气 |
| 全脚掌着地 | 增加稳定性，减少小腿疲劳 |
| 使用登山杖 | 分担腿部压力，节省 20-30% 体力 |
| 之字形攀登 | 陡坡走之字，降低坡度 |
| 调整呼吸 | 深呼吸，节奏均匀 |

#### 下坡技巧

| 要点 | 说明 |
|------|------|
| 重心降低 | 微屈膝，重心稍向后 |
| 小步试探 | 先探路，确认稳固再落脚 |
| 侧身下坡 | 陡坡可侧身，增加稳定性 |
| 使用登山杖 | 减轻膝盖冲击 |
| 避免奔跑 | 容易失控摔倒 |

#### 休息原则

| 原则 | 说明 |
|------|------|
| 短频休息 | 每 30-45 分钟休息 5-10 分钟 |
| 不要坐地 | 站立或靠树，避免身体冷却 |
| 及时补水 | 少量多次，每次 100-200ml |
| 补充能量 | 休息时吃少量高能量食物 |
| 注意保暖 | 休息时加衣，避免着凉 |

### 2.2 补水与营养

#### 补水原则

| 时间 | 补水量 | 说明 |
|------|--------|------|
| 登山前 1 小时 | 300-500ml | 提前补水 |
| 行进中 | 150-250ml/30 分钟 | 少量多次 |
| 休息时 | 200-300ml | 及时补充 |
| 登山后 | 500-1000ml | 充分补充 |

**注意事项**：
- ⚠️ 不要等口渴了再喝水
- ⚠️ 避免一次性大量饮水
- ⚠️ 高海拔地区需增加饮水量
- ⚠️ 冬季也要补水（易忽视）

#### 能量补充

| 食物类型 | 推荐 | 说明 |
|---------|------|------|
| 碳水化合物 | 面包、饼干、能量棒 | 快速供能 |
| 蛋白质 | 坚果、牛肉干、鸡蛋 | 持续供能 |
| 电解质 | 运动饮料、盐丸 | 防抽筋 |
| 糖分 | 巧克力、葡萄糖 | 快速补充 |

**建议携带**：
- 路餐：三明治、饭团、面包
- 零食：坚果、葡萄干、能量棒
- 应急：巧克力、葡萄糖片

### 2.3 安全要点

#### 行走安全

| 要点 | 说明 |
|------|------|
| 保持距离 | 前后队员间隔 3-5 米 |
| 注意落石 | 陡坡下方快速通过，不逗留 |
| 防滑防摔 | 湿滑路面小心，必要时手脚并用 |
| 远离边缘 | 悬崖边保持安全距离 |
| 不抄近道 | 沿既定路线，避免迷路 |

#### 野生动物

| 动物 | 应对方法 |
|------|---------|
| 蛇 | 保持距离，用登山杖敲打前方草丛 |
| 野猪 | 慢慢后退，不要挑衅 |
| 猴子 | 不喂食，食物收好 |
| 熊 | 慢慢后退，不要奔跑，制造声响 |
| 马蜂 | 远离蜂巢，被蛰后及时就医 |

#### 天气突变应对

| 情况 | 应对措施 |
|------|---------|
| 雷雨 | 立即下撤，远离高处、树木、水体 |
| 大雾 | 停止前进，原地等待或下撤 |
| 暴雨 | 找安全处躲避，防范山洪 |
| 大风 | 降低重心，寻找避风处 |
| 降雪 | 注意保暖，防范失温 |

## 三、高海拔登山特殊注意事项

### 3.1 高原反应

#### 症状识别

| 程度 | 症状 | 应对 |
|------|------|------|
| 轻度 | 头痛、失眠、食欲减退 | 休息、补水、适应 |
| 中度 | 头痛加重、恶心、呼吸困难 | 停止上升、下撤 300-500m |
| 重度 | 剧烈头痛、呕吐、意识模糊 | 立即下撤、就医 |

#### 预防措施

| 措施 | 说明 |
|------|------|
| 循序渐进 | 每日上升不超过 500m（3000m 以上） |
| 充分适应 | 每上升 1000m 适应 1-2 天 |
| 多喝水 | 高海拔需增加饮水量 |
| 不饮酒 | 酒精加重高原反应 |
| 药物预防 | 红景天（提前服用）、乙酰唑胺 |

### 3.2 高海拔装备

| 装备 | 说明 |
|------|------|
| 保暖衣物 | 羽绒服、保暖帽、手套 |
| 防晒装备 | 高倍防晒霜、雪镜 |
| 氧气瓶 | 应急用（4000m 以上建议携带） |
| 血氧仪 | 监测血氧饱和度 |
| 高山靴 | 4000m 以上需要 |

## 四、常见风险与应对

### 4.1 常见伤病

| 伤病 | 预防 | 处理 |
|------|------|------|
| 扭伤 | 穿合适的鞋，使用登山杖 | RICE 原则（休息、冰敷、加压、抬高） |
| 擦伤 | 穿长裤，注意路况 | 清洗消毒，包扎 |
| 水泡 | 穿合适的鞋袜，贴防磨贴 | 消毒后挑破，包扎 |
| 抽筋 | 充分热身，补充电解质 | 拉伸按摩，补充盐分 |
| 晒伤 | 防晒霜，物理防晒 | 冷敷，芦荟胶 |
| 失温 | 及时加衣，保持干燥 | 换干衣，保暖，热饮 |
| 中暑 | 避开高温时段，多喝水 | 阴凉处休息，降温，补水 |

### 4.2 迷路应对

#### 预防措施

| 措施 | 说明 |
|------|------|
| 下载离线地图 | 两步路、六只脚等 APP |
| 标记轨迹 | 记录行进路线 |
| 注意路标 | 留意路牌、标记 |
| 不抄近道 | 沿既定路线行走 |
| 告知行踪 | 出发前告知亲友行程 |

#### 迷路后处理

| 步骤 | 行动 |
|------|------|
| 1. 停止前进 | 发现迷路立即停止 |
| 2. 冷静分析 | 回忆路线，查看地图 |
| 3. 尝试返回 | 沿原路返回已知位置 |
| 4. 求救 | 拨打救援电话，发送定位 |
| 5. 等待救援 | 找安全处，保存体力 |

**求救电话**：
- 报警：110
- 急救：120
- 消防救援：119
- 山地救援：当地救援队电话

### 4.3 失温与中暑

#### 失温（低温症）

| 程度 | 症状 | 处理 |
|------|------|------|
| 轻度 | 发抖、手脚冰凉 | 加衣、热饮、运动产热 |
| 中度 | 发抖停止、意识模糊 | 换干衣、保温毯、外部加热 |
| 重度 | 昏迷、脉搏微弱 | 立即就医、持续保温 |

**预防**：
- 带足保暖衣物
- 保持身体干燥
- 及时补充热量
- 避免大风中休息

#### 中暑

| 程度 | 症状 | 处理 |
|------|------|------|
| 先兆 | 头晕、恶心、多汗 | 阴凉处休息、补水 |
| 轻度 | 体温升高、面色潮红 | 降温、补水、电解质 |
| 重度 | 高热、意识障碍 | 立即就医、物理降温 |

**预防**：
- 避开高温时段（10:00-15:00）
- 戴遮阳帽、穿浅色衣物
- 多喝水、补充电解质
- 适当休息

## 五、环保与礼仪

### 5.1 无痕登山原则

| 原则 | 说明 |
|------|------|
| 带走垃圾 | 所有垃圾带下山，包括果皮 |
| 不破坏植被 | 不采摘花草，不砍伐树木 |
| 不惊扰动物 | 不追逐、不喂食野生动物 |
| 不污染水源 | 不在水源处使用化学用品 |
| 不留下痕迹 | 不刻字、不堆石堆 |

### 5.2 登山礼仪

| 场景 | 礼仪 |
|------|------|
| 狭窄路段 | 上坡优先，下坡让行 |
| 休息时 | 不占用步道，靠边休息 |
| 拍照时 | 不阻碍他人通行 |
| 声音控制 | 不大声喧哗，保持安静 |
| 营地选择 | 不占用他人营地，保持距离 |
