## 一、概述

### 1.1 尿素简介

**尿素 (Urea)**：化学式 CO(NH₂)₂，分子量 60.06

| 属性 | 数值/说明 |
|------|----------|
| 含氮量 | 46.4%（固体氮肥中最高） |
| 熔点 | 132.7°C |
| 密度 | 1.335 g/cm³ |
| 水溶性 | 易溶（20°C 时 108 g/100ml） |
| 全球产量 | 约 2 亿吨/年（2025 年） |
| 中国产量 | 约 7000 万吨/年（2025 年） |

### 1.2 主要用途

| 用途 | 占比 | 说明 |
|------|------|------|
| **农业肥料** | 85-90% | 固体氮肥、液体肥、缓释肥 |
| **工业应用** | 5-8% | 塑料、树脂、胶粘剂 |
| **车用尿素** | 3-5% | 柴油车尾气处理 (SCR) |
| **其他** | 2-3% | 医药、食品、饲料等 |

### 1.3 产业链位置

```
天然气/煤炭 → 合成氨 (NH₃) → 尿素 CO(NH₂)₂
                          ↓
                    二氧化碳 (CO₂)
```

**原料消耗**（以 1 吨尿素计）：
- 合成氨：0.57 吨
- 二氧化碳：0.75 吨（通常来自合成氨厂副产）

## 二、化学反应原理

### 2.1 总反应式

```
2NH₃(l) + CO₂(g) → CO(NH₂)₂(s) + H₂O(l)  ΔH = -103.7 kJ/mol
```

### 2.2 反应机理（两步法）

#### 第一步：氨基甲酸铵生成（快速、强放热）
```
2NH₃ + CO₂ → NH₂COONH₄  (氨基甲酸铵)
ΔH₁ = -159 kJ/mol
```

| 条件 | 数值 |
|------|------|
| 温度 | 80-120°C |
| 压力 | 15-25 MPa |
| 反应速率 | 快（秒级完成） |
| 转化率 | 接近 100% |

#### 第二步：尿素生成（慢速、吸热、可逆）
```
NH₂COONH₄ → CO(NH₂)₂ + H₂O
ΔH₂ = +55.3 kJ/mol
```

| 条件 | 数值 |
|------|------|
| 温度 | 180-210°C |
| 压力 | 14-20 MPa |
| 反应速率 | 慢（需 30-60 分钟） |
| 平衡转化率 | 50-70% |

### 2.3 反应平衡

**影响平衡转化率的因素**：

| 因素 | 影响 | 优化方向 |
|------|------|---------|
| 温度 | 温度↑ → 转化率↑（但过高会分解） | 180-210°C 最佳 |
| NH₃/CO₂ 摩尔比 | 比值↑ → 转化率↑ | 3.0-4.0（过量氨） |
| H₂O/CO₂ 摩尔比 | 比值↑ → 转化率↓ | 尽量降低水含量 |
| 压力 | 压力↑ → 转化率↑ | 14-20 MPa |

## 三、工艺流程

### 3.1 典型工艺流程图

```
                    ┌─────────────┐
     液氨 → 高压泵 →│             │
                    │   合成塔    │ → 反应液（尿素 + 甲铵 + 未反应物）
     CO₂ → 压缩机 →│             │
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │   高压分解  │ → 未反应 NH₃/CO₂ 回收
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │   中压分解  │ → 进一步回收
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │   低压分解  │ → 深度回收
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │   浓缩蒸发  │ → 水分蒸发
                    └──────┬──────┘
                           ↓
                    ┌─────────────┐
                    │   造粒塔    │ → 尿素颗粒
                    └─────────────┘
```

### 3.2 主要工艺步骤

#### （1）原料压缩与输送

| 物料 | 状态 | 压力 | 温度 |
|------|------|------|------|
| 液氨 | 液态 | 2.0 MPa → 20 MPa | 常温 |
| CO₂ | 气态 | 0.02 MPa → 20 MPa | 常温 → 40°C |

**关键设备**：
- 液氨高压泵（往复泵或离心泵）
- CO₂ 压缩机（多级离心式或往复式）

#### （2）合成反应

**设备**：尿素合成塔（立式高压容器）

| 参数 | 数值 |
|------|------|
| 材质 | 316L 不锈钢（内衬） |
| 尺寸 | Φ2000-4000mm × H20000-40000mm |
| 设计压力 | 22-25 MPa |
| 操作温度 | 180-210°C |
| 停留时间 | 30-60 分钟 |
| CO₂ 转化率 | 58-65% |

**塔内结构**：
```
┌─────────────────┐
│   气体分布器    │ ← CO₂ 入口
├─────────────────┤
│   筛板/填料     │ ← 气液混合
├─────────────────┤
│   反应区        │ ← 主要反应区域
├─────────────────┤
│   液体收集器    │ ← 反应液出口
└─────────────────┘
       ↑
    液氨入口
```

#### （3）未反应物回收

**目的**：分离并回收未反应的 NH₃ 和 CO₂，循环使用

**三级分解回收系统**：

| 级别 | 压力 | 温度 | 回收率 |
|------|------|------|--------|
| 高压分解 | 1.7-2.0 MPa | 160-170°C | NH₃ 70%, CO₂ 60% |
| 中压分解 | 0.3-0.5 MPa | 140-150°C | NH₃ 20%, CO₂ 25% |
| 低压分解 | 0.03-0.05 MPa | 120-130°C | NH₃ 8%, CO₂ 12% |

**总回收率**：NH₃ >98%, CO₂ >97%

#### （4）尿液浓缩

**设备**：真空蒸发器

| 参数 | 进料 | 出料 |
|------|------|------|
| 尿素浓度 | 70-75% | 99.5-99.7% |
| 温度 | 130-140°C | 135-140°C |
| 压力 | 常压 | 真空 (0.03 MPa) |

**关键控制**：
- 温度不能过高（防止尿素分解）
- 停留时间尽量短
- 真空度稳定

#### （5）造粒

**两种主流技术**：

##### A. 造粒塔喷淋法（传统）
```
    熔融尿素 (140°C)
         ↓
    旋转喷头
         ↓
    液滴下落 (100m 高塔)
         ↓
    冷却固化 → 颗粒 (1-4mm)
         ↓
    筛分 → 成品
```

| 参数 | 数值 |
|------|------|
| 塔高 | 80-120m |
| 颗粒直径 | 1-4mm |
| 含水量 | <0.5% |
| 抗压强度 | >50N/颗粒 |

**优点**：技术成熟、投资低  
**缺点**：粉尘大、颗粒强度低、易结块

##### B. 流化床造粒（先进）
```
    熔融尿素 → 喷雾 → 流化床
                        ↓
                   热空气流化
                        ↓
                   冷却 + 包裹
                        ↓
                   成品颗粒
```

| 参数 | 数值 |
|------|------|
| 颗粒直径 | 2-4mm（可调） |
| 含水量 | <0.3% |
| 抗压强度 | >100N/颗粒 |
| 粉尘 | 极少 |

**优点**：颗粒强度高、粉尘少、可生产大颗粒  
**缺点**：投资高、操作复杂

## 九、绿氢耦合尿素生产

### 9.1 技术路线

```
绿电 → 电解水 → H₂
              ↓
         合成氨（绿氨）
              ↓
         尿素合成
              ↓
         绿色尿素
```

**关键变化**：
- 合成氨由绿电制氢 + 空分氮气生产
- CO₂ 可来自生物质或碳捕集
- 产品为"绿色尿素"，碳足迹大幅降低

### 9.2 碳足迹对比

| 产品 | 碳排放（吨 CO₂/吨尿素） |
|------|----------------------|
| 煤制尿素 | 2.5-3.0 |
| 天然气制尿素 | 1.2-1.5 |
| 绿氢尿素（绿电 + 捕集 CO₂） | 0.1-0.3 |

### 9.3 经济性

| 项目 | 传统尿素 | 绿氢尿素 |
|------|---------|---------|
| 成本 | 2400-3200 元/吨 | 4500-6000 元/吨 |
| 售价 | 2500-3500 元/吨 | 溢价 30-50% |
| 关键因素 | 煤炭/天然气价格 | 绿电成本 |

**盈亏平衡点**：绿电 <0.2 元/kWh 时具备竞争力

### 9.4 应用前景
