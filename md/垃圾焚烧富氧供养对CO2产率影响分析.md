## 一、富氧燃烧技术概述

### 1.1 什么是富氧燃烧

**定义**：使用氧气浓度高于空气（21%）的气体作为助燃剂

| 类型 | 氧气浓度 | 说明 |
|------|---------|------|
| 空气燃烧 | 21% O2 | 常规方式 |
| 低浓度富氧 | 25-30% O2 | 轻度富氧 |
| 中浓度富氧 | 30-50% O2 | 中度富氧 |
| 高浓度富氧 | 50-95% O2 | 深度富氧 |
| 纯氧燃烧 | >95% O2 | 接近纯氧 |

### 1.2 富氧燃烧原理

**常规空气燃烧**：
```
垃圾 + 空气（21% O2 + 79% N2） → 烟气 + 灰渣 + 热能
烟气成分：CO2(8-12%) + H2O(15-25%) + N2(55-65%) + O2(6-10%)
```

**富氧燃烧**：
```
垃圾 + 富氧空气（30-50% O2） → 烟气 + 灰渣 + 热能
烟气成分：CO2(15-25%) + H2O(20-30%) + N2(30-50%) + O2(3-8%)
```

**纯氧燃烧**：
```
垃圾 + 纯氧（>95% O2） → 烟气 + 灰渣 + 热能
烟气成分：CO2(60-80%) + H2O(20-35%) + 少量杂质
```

## 二、富氧对 CO2 产率的影响

### 2.1 CO2 浓度变化

| 富氧浓度 | 烟气 CO2 浓度 | 提升倍数 |
|---------|-------------|---------|
| 空气（21% O2） | 8-12% | 基准 |
| 25% O2 | 12-16% | 1.3-1.5 倍 |
| 30% O2 | 15-20% | 1.5-1.8 倍 |
| 40% O2 | 20-30% | 2.0-2.5 倍 |
| 50% O2 | 25-40% | 2.5-3.5 倍 |
| 纯氧（95% O2） | 60-80% | 5-7 倍 |

**关键结论**：
- ✅ **CO2 浓度显著提高**（不是产率，是浓度）
- ⚠️ **CO2 总产量基本不变**（由垃圾碳含量决定）
- ✅ **烟气总量大幅减少**（氮气减少）

### 2.2 CO2 产生量（质量）

**CO2 产生量由垃圾碳含量决定，与富氧无关**：

| 垃圾成分 | 含碳量 | CO2 产生量 |
|---------|--------|-----------|
| 厨余垃圾 | 15-20% | 0.55-0.73 吨 CO2/吨垃圾 |
| 纸张 | 35-40% | 1.28-1.47 吨 CO2/吨垃圾 |
| 塑料 | 60-75% | 2.20-2.75 吨 CO2/吨垃圾 |
| 纺织品 | 40-50% | 1.47-1.83 吨 CO2/吨垃圾 |
| 混合垃圾（典型） | 25-35% | 0.92-1.28 吨 CO2/吨垃圾 |

**实际 CO2 产生量**（考虑燃烧效率）：
| 条件 | CO2 产生量 |
|------|-----------|
| 空气燃烧 | 0.5-0.8 吨 CO2/吨垃圾 |
| 富氧燃烧 | 0.55-0.85 吨 CO2/吨垃圾 |
| 提升 | 约 5-10%（燃烧更完全） |

### 2.3 烟气量变化

| 富氧浓度 | 烟气量 | 对比空气燃烧 |
|---------|--------|-------------|
| 空气（21% O2） | 3500-5000 Nm³/吨 | 基准 |
| 30% O2 | 2500-3500 Nm³/吨 | ↓ 25-30% |
| 40% O2 | 1800-2500 Nm³/吨 | ↓ 45-50% |
| 50% O2 | 1200-1800 Nm³/吨 | ↓ 60-65% |
| 纯氧 | 800-1200 Nm³/吨 | ↓ 75-80% |

**关键优势**：烟气量减少意味着：
- 后续处理设备规模减小
- CO2 捕集能耗降低
- 排放污染物总量减少

### 2.4 烟气成分对比

| 组分 | 空气燃烧 | 30% 富氧 | 50% 富氧 | 纯氧燃烧 |
|------|---------|---------|---------|---------|
| CO2 | 8-12% | 15-20% | 25-40% | 60-80% |
| H2O | 15-25% | 20-30% | 25-35% | 20-35% |
| N2 | 55-65% | 35-50% | 20-35% | 1-5% |
| O2 | 6-10% | 3-8% | 3-5% | 2-5% |
| 污染物 | 基准 | ↓20-30% | ↓40-50% | ↓60-80% |

## 三、富氧燃烧的技术优势

### 3.1 燃烧效率提升

| 指标 | 空气燃烧 | 富氧燃烧 | 提升 |
|------|---------|---------|------|
| 燃烧温度 | 850-1000°C | 1000-1200°C | ↑150-200°C |
| 燃尽率 | 95-97% | 97-99% | ↑2-3% |
| 过量空气系数 | 1.6-2.0 | 1.2-1.5 | ↓25-30% |
| 热灼减率 | 3-5% | 1-3% | ↓40-50% |

**效果**：
- 垃圾燃烧更完全
- 残渣含碳量降低
- CO2 产生量略有增加（5-10%）

### 3.2 污染物减排

| 污染物 | 空气燃烧 | 富氧燃烧 | 减排 |
|--------|---------|---------|------|
| NOx | 200-300 mg/m³ | 100-200 mg/m³ | ↓30-50% |
| CO | 50-100 mg/m³ | 20-50 mg/m³ | ↓50-60% |
| 二噁英 | 0.05-0.1 ng TEQ/m³ | 0.02-0.05 ng TEQ/m³ | ↓50-60% |
| 粉尘 | 10-20 mg/m³ | 5-10 mg/m³ | ↓50% |

**原因**：
- 燃烧温度高且均匀
- 过量空气少，热力型 NOx 减少
- 燃烧完全，CO 和未燃尽有机物减少

### 3.3 对 CO2 捕集的影响

**富氧燃烧使 CO2 捕集更容易**：

| 指标 | 空气燃烧 | 30% 富氧 | 50% 富氧 |
|------|---------|---------|---------|
| CO2 浓度 | 8-12% | 15-20% | 25-40% |
| 烟气量 | 100% | 70-75% | 35-40% |
| 捕集能耗 | 基准 | ↓20-30% | ↓40-50% |
| 设备规模 | 基准 | ↓25-30% | ↓50-60% |
| 捕集成本 | 基准 | ↓15-25% | ↓30-40% |

## 八、总结与建议

### 8.1 技术结论

✅ **富氧燃烧可显著提高烟气 CO2 浓度**（2-3 倍）  
✅ **有利于后续 CO2 捕集**（能耗降低 20-40%）  
⚠️ **CO2 总产量变化不大**（略增 5-10%）  
✅ **燃烧效率提升，污染物减排**

### 8.2 经济结论

| 条件 | 经济性 |
|------|--------|
| 配套 CO2 捕集 | ✅ 经济可行（回收期<1 年） |
| 无 CO2 捕集 | ❌ 经济不可行（年亏损 100-300 万元） |

| 有政府补贴 | ✅ 经济可行 |

### 8.3 实施建议

| 企业类型 | 建议 |
|---------|------|
| 新建大型厂 | 预留富氧接口，根据碳政策决定 |
