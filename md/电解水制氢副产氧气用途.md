## 一、副产氧气的基本数据

### 1.1 产生量

电解水制氢的化学反应：
```
2H₂O → 2H₂ + O₂
```

**质量比**：每生产 **1 kg 氢气**，副产 **8 kg 氧气**

**体积比**：每生产 **1 Nm³ 氢气**，副产 **0.5 Nm³ 氧气**

### 1.2 典型项目副产量

| 项目规模 | 氢气产量 | 氧气副产量 |
|---------|---------|-----------|
| 10 MW 电解槽 | 约 2 吨/天 H₂ | 约 16 吨/天 O₂ |
| 100 MW 电解槽 | 约 20 吨/天 H₂ | 约 160 吨/天 O₂ |
| 1 GW 电解槽 | 约 200 吨/天 H₂ | 约 1600 吨/天 O₂ |

### 1.3 氧气纯度

| 电解技术 | 氧气纯度 |
|---------|---------|
| 碱性电解槽 (ALK) | 99.0-99.5% |
| 质子交换膜 (PEM) | 99.5-99.9% |
| 固体氧化物 (SOEC) | 99.0-99.5% |

## 二、氧气的主要用途

### 2.1 工业应用（最大市场）

#### （1）钢铁冶炼
| 用途 | 说明 | 耗氧量 |
|------|------|--------|
| 高炉富氧鼓风 | 提高燃烧效率，增加产量 | 20-50 Nm³/吨钢 |
| 转炉炼钢 | 氧化去除碳、磷等杂质 | 50-80 Nm³/吨钢 |
| 电炉炼钢 | 辅助熔化，提高效率 | 10-30 Nm³/吨钢 |

**市场潜力**：中国年产钢约 10 亿吨，理论需氧量超 500 亿 Nm³/年

#### （2）金属切割与焊接
- **氧炔切割**：氧气 + 乙炔，用于厚板切割
- **氧焊**：氧气助燃，提高火焰温度
- **市场规模**：相对较小，但需求稳定

#### （3）化工氧化剂
| 应用 | 说明 |
|------|------|
| 乙烯氧化制环氧乙烷 | 重要化工原料 |
| 丙烯氧化制丙烯酸 | 塑料、涂料原料 |
| 对二甲苯氧化制 PTA | 聚酯纤维原料 |
| 煤气化 | 生产合成气 (CO+H₂) |

### 2.2 环保应用

#### （1）污水处理
| 用途 | 说明 |
|------|------|
| 曝气增氧 | 提高好氧微生物活性，加速有机物分解 |
| 臭氧制备 | 氧气制备臭氧，用于深度处理和消毒 |
| 污泥处理 | 好氧消化，减少污泥体积 |

**优势**：相比空气曝气，纯氧曝气效率提高 2-3 倍

#### （2）废气处理
- **焚烧炉富氧燃烧**：提高燃烧温度，彻底分解有害物质
- **脱硫脱硝**：辅助氧化反应

### 2.3 医疗用途

| 应用 | 说明 | 要求 |
|------|------|------|
| 医院供氧 | 手术室、病房、ICU 等 | 医用氧标准 (≥99.5%) |
| 家用氧疗 | 慢性呼吸系统疾病患者 | 需进一步纯化 |
| 高压氧舱 | 一氧化碳中毒、创伤治疗等 | 高纯度要求 |

**注意**：医用氧需要单独认证，生产设备需符合 GMP 要求

### 2.4 其他应用

| 领域 | 用途 |
|------|------|
| **水产养殖** | 增氧机、运输增氧，提高养殖密度 |
| **玻璃制造** | 富氧燃烧，提高熔化效率，减少 NOx 排放 |
| **造纸** | 纸浆漂白，减少氯气使用 |
| **半导体** | 氧化工艺，高纯度要求 (99.999%+) |
| **航空航天** | 生命保障系统、火箭氧化剂 |

## 三、绿氢项目氧气利用现状

### 3.1 当前处理方式

| 处理方式 | 占比 | 说明 |
|---------|------|------|
| 直接排放 | 60-70% | 大多数项目选择 |
| 工业销售 | 20-30% | 附近有用户需求 |
| 综合利用 | 10-20% | 污水处理、医疗等 |

### 3.2 典型案例

#### 案例 1：内蒙古鄂尔多斯绿氢项目
- **规模**：300 MW 光伏 + 30 MW 电解水制氢
- **氢气用途**：煤化工耦合
- **氧气用途**：直接排放（周边无大型用氧企业）

#### 案例 2：吉林松原绿氢甲醇项目
- **规模**：500 MW 风光 + 50 MW 电解水
- **氢气用途**：绿色甲醇合成
- **氧气用途**：部分用于生物质气化富氧，其余排放

#### 案例 3：河北张家口绿氢项目
- **规模**：10 MW PEM 电解槽
- **氢气用途**：冬奥会燃料电池汽车
- **氧气用途**：供应附近医院和污水处理厂

## 五、氧气利用的挑战

### 5.1 市场容量限制

- **区域市场饱和**：传统空分制氧已满足大部分工业需求
- **运输半径**：气态氧经济运输半径约 200-300 km
- **需求波动**：工业用氧需求相对稳定，但增长有限

### 5.2 技术挑战

| 挑战 | 说明 |
|------|------|
| 纯度波动 | 电解槽负荷变化影响氧气纯度 |
| 连续供应 | 绿电波动导致氧气供应不稳定 |
| 储存成本 | 需要配置储罐或液化设备 |
| 安全要求 | 氧气易燃易爆，需要严格安全管理 |

### 5.3 经济性挑战

- **投资成本**：压缩、储存、纯化设备需要额外投资
- **运营成本**：压缩和运输增加能耗
- **价格竞争**：与成熟的空分制氧竞争

## 六、优化利用建议

### 6.1 项目规划阶段

1. **选址考虑**：优先选择靠近用氧企业的地区
   - 钢铁厂、化工厂附近
   - 大型医院集中区域
   - 污水处理厂附近

2. **规模匹配**：根据周边氧气需求确定电解槽规模

3. **一体化设计**：将氧气利用纳入项目整体设计

### 6.2 技术路线选择

| 方案 | 适用场景 | 投资 |
|------|---------|------|
| 直接管道供应 | 附近有稳定用户 | 低 |
| 压缩储存 + 槽车运输 | 中距离多用户 | 中 |
| 液化储存 | 远距离或大量储存 | 高 |
| 现场利用 | 污水处理、生物质气化等 | 中 |

### 6.3 商业模式创新

1. **氧气置换**：与空分企业交换，由空分企业供应周边用户，绿氢项目供应空分企业电力

2. **合同能源管理**：与用氧企业签订长期供应合同

3. **碳减排认证**：将氧气利用纳入碳减排量计算

## 七、未来发展方向

### 7.1 技术创新

- **高效压缩技术**：降低压缩能耗
- **小型液化设备**：降低储存和运输成本

- **出口市场**：液氧出口到氧气短缺地区

### 7.3 政策支持

- **纳入绿氢标准**：将氧气利用效率纳入绿氢项目评价体系
- **财政补贴**：对氧气综合利用项目给予补贴
