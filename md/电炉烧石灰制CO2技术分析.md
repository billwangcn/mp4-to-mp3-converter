## 一、基本原理

### 1.1 化学反应

石灰石煅烧分解反应：
```
CaCO₃ → CaO + CO₂↑  (吸热反应)
```

**热力学参数**：
| 参数 | 数值 |
|------|------|
| 反应温度 | 850-950°C |
| 反应热 | +178 kJ/mol (吸热) |
| 理论能耗 | 约 3.2 GJ/吨 CaO |
| CO₂ 产生量 | 0.78 吨 CO₂/吨 石灰石 |

### 1.2 物料平衡

**以 1 吨石灰石为例**：
| 输入 | 输出 |
|------|------|
| 石灰石 (CaCO₃) 1 吨 | 生石灰 (CaO) 0.56 吨 |
| - | CO₂ 0.44 吨 |

**以 1 吨生石灰为例**：
| 输入 | 输出 |
|------|------|
| 石灰石 1.78 吨 | 生石灰 (CaO) 1 吨 |
| - | CO₂ 0.78 吨 |

## 二、电炉煅烧技术路线

### 2.1 电加热方式

| 方式 | 原理 | 温度范围 | 效率 |
|------|------|---------|------|
| **电阻加热** | 电热元件辐射加热 | 900-1100°C | 70-85% |
| **电弧加热** | 电极间电弧高温 | 1200-2000°C | 60-75% |
| **感应加热** | 电磁感应涡流加热 | 800-1200°C | 75-90% |
| **等离子加热** | 等离子体高温加热 | 2000-5000°C | 50-70% |

### 2.2 炉型选择

#### （1）回转窑（电加热）
```
┌─────────────────────────────────────────┐
│    进料 → ═══════════════→ 出料        │
│         电加热区 (900-950°C)            │
└─────────────────────────────────────────┘
```
| 参数 | 典型值 |
|------|--------|
| 产能 | 50-500 吨/天 |
| 热效率 | 70-80% |
| 电耗 | 800-1200 kWh/吨 CaO |
| 优点 | 连续生产、产量大 |
| 缺点 | 设备投资高 |

#### （2）竖窑（电加热）
```
    进料
     ↓
┌─────────┐
│ 预热区  │
├─────────┤
│ 煅烧区  │ ← 电加热
├─────────┤
│ 冷却区  │
└─────────┘
     ↓
    出料
```
| 参数 | 典型值 |
|------|--------|
| 产能 | 20-200 吨/天 |
| 热效率 | 75-85% |
| 电耗 | 700-1000 kWh/吨 CaO |
| 优点 | 热效率高、投资较低 |
| 缺点 | 对原料粒度要求高 |

#### （3）流化床电炉
| 参数 | 典型值 |
|------|--------|
| 产能 | 10-100 吨/天 |
| 热效率 | 80-90% |
| 电耗 | 650-900 kWh/吨 CaO |
| 优点 | 传热效率高、温度均匀 |
| 缺点 | 操作复杂、粉尘多 |

### 2.3 推荐方案

**对于 CO2 捕集目的**，推荐**密闭式电加热竖窑**：
- 烟气中 CO2 浓度高（>95%），便于捕集
- 热效率较高
- 投资适中

## 三、CO2 捕集与纯化

### 3.1 烟气成分

| 成分 | 电炉煅烧 | 煤煅烧 | 天然气煅烧 |
|------|---------|--------|-----------|
| CO₂ | 95-98% | 20-30% | 15-25% |
| N₂ | 1-3% | 60-70% | 5-10% |
| O₂ | 1-2% | 3-5% | 3-5% |
| H₂O | 0-1% | 5-8% | 10-15% |

**关键优势**：电炉煅烧烟气中 CO2 浓度远高于燃烧法，捕集成本大幅降低

### 3.2 纯化工艺

```
窑气 → 除尘 → 冷却脱水 → 压缩 → 吸附纯化 → 液化 → 液 CO₂
```

| 工序 | 目的 | 指标 |
|------|------|------|
| 除尘 | 去除粉尘 | <10 mg/Nm³ |
| 冷却脱水 | 去除水分 | 露点 <-40°C |
| 压缩 | 提高压力 | 2-3 MPa |
| 吸附纯化 | 去除杂质 | CO₂ >99.9% |
| 液化 | 便于储存运输 | -20°C 至 -30°C |

### 3.3 产品规格

| 等级 | 纯度 | 用途 |
|------|------|------|
| 工业级 | 99.5-99.9% | 焊接、化工、水处理 |
| 食品级 | 99.9%+ | 饮料、食品保鲜 |
| 电子级 | 99.999%+ | 半导体制造 |

## 七、技术挑战

### 7.1 主要挑战

| 挑战 | 说明 | 解决方向 |
|------|------|---------|
| **电耗高** | 电加热成本远高于燃料 | 使用低价绿电、提高热效率 |
| **电极寿命** | 高温下电极易损耗 | 选用耐高温材料、优化设计 |
| **温度控制** | 需精确控制 900-950°C | 智能控制系统 |
| **CO2 纯化** | 食品级要求高 | 多级纯化工艺 |
| **经济性** | 整体成本偏高 | 规模化、绿电补贴 |

### 7.2 与绿氢耦合方案

```
绿电 → 电解水制氢 → 氢气
     → 电炉烧石灰 → CO2
                    ↓
              绿氢 + CO2 → 绿色甲醇
```

**优势**：
- 同时获得 H2 和 CO2 原料
- 共用绿电基础设施
- 产品附加值高

## 九、结论与建议

### 9.1 技术可行性

✅ **技术上完全可行**
- 电加热石灰煅烧是成熟技术
- CO2 捕集纯化工工艺成熟
- 可生产工业级和食品级 CO2

### 9.2 经济可行性

⚠️ **取决于关键条件**

| 条件 | 要求 | 现状 |
|------|------|------|
| 电价 | <0.5 元/kWh | 绿电基地可达 |
| CO2 价格 | >400 元/吨 | 食品级可达 |
| 规模 | >50 吨/天 | 可实现 |
| 石灰石 | 就近供应 | 资源丰富 |

### 9.3 推荐方案

**最佳应用场景**：
1. **绿电基地 + 食品级 CO2**：电价低 + 产品附加值高
2. **绿氢甲醇耦合**：同时生产 H2 和 CO2，用于绿色甲醇合成
3. **工业园区配套**：就近供应 CO2 用户

### 9.4 不建议场景

- 电价 >0.8 元/kWh 地区
- 远离 CO2 用户（运输成本高）
- 小规模生产（<20 吨/天）
