# 垃圾焚烧厂废气 CO2 回收利用分析

**信息来源**: 公开技术资料、行业报告、学术论文

## 一、垃圾焚烧厂废气概况

### 1.1 垃圾焚烧工艺简介

```
生活垃圾 → 进料 → 焚烧炉（850-1100°C） → 热能 → 发电/供热
                    ↓
                烟气系统
                    ↓
        脱硝 → 除尘 → 脱酸 → 活性炭 → 布袋除尘 → 烟囱排放
```

### 1.2 烟气成分与排放量

**典型烟气成分（体积浓度）**：

| 组分 | 浓度范围 | 说明 |
|------|---------|------|
| N₂ | 55-65% | 主要来自空气 |
| CO₂ | 8-12% | 垃圾中有机物燃烧产生 |
| H₂O | 15-25% | 垃圾水分蒸发 |
| O₂ | 6-10% | 过量空气 |
| CO | <50 ppm | 不完全燃烧产物 |
| NOx | 100-300 ppm | 需脱硝处理 |
| SO₂ | 50-200 ppm | 需脱酸处理 |
| HCl | 50-500 ppm | 塑料等含氯物质燃烧 |
| 二噁英 | <0.1 ng TEQ/m³ | 严格控制 |
| 粉尘 | <10 mg/m³ | 布袋除尘后 |

**烟气排放量**：

| 指标 | 数值 |
|------|------|
| 吨垃圾烟气量 | 3500-5000 Nm³/吨垃圾 |
| CO2 产生量 | 0.5-0.8 吨 CO2/吨垃圾 |
| 年处理 100 万吨垃圾厂 | 排放 CO2 约 50-80 万吨/年 |

### 1.3 CO2 来源构成

垃圾焚烧产生的 CO2 来自：

| 来源 | 占比 | 性质 |
|------|------|------|
| 生物质（厨余、纸张、木材） | 50-60% | 生物源 CO2（碳中性） |
| 化石源（塑料、化纤） | 40-50% | 化石源 CO2（需减排） |

**关键区别**：
- 生物源 CO2：不计入碳排放（碳循环）
- 化石源 CO2：计入碳排放（新增碳）

### 1.4 与燃煤/燃气电厂对比

| 指标 | 垃圾焚烧 | 燃煤电厂 | 燃气电厂 |
|------|---------|---------|---------|
| CO2 浓度 | 8-12% | 12-15% | 5-8% |
| 烟气量 | 大 | 中 | 中 |
| 排放因子 | 0.5-0.8 吨/吨垃圾 | 2.5-3.0 吨/吨煤 | 1.5-2.0 吨/吨气 |
| 排放稳定性 | 稳定 | 稳定 | 稳定 |
| 捕集难度 | 中等 | 中等 | 中等 |

## 二、CO2 回收技术适用性

### 2.1 技术路线对比

| 技术 | 适用浓度 | 垃圾焚烧适用性 | 能耗 | 投资 |
|------|---------|--------------|------|------|
| **化学吸收法** | 5-30% | ⭐⭐⭐⭐⭐ | 高 | 中 |
| **物理吸收法** | >30% | ⭐⭐ | 中 | 中 |
| **变压吸附** | 10-50% | ⭐⭐⭐ | 中 | 低 |
| **膜分离** | >20% | ⭐⭐ | 低 | 低 |
| **低温蒸馏** | >50% | ⭐ | 高 | 高 |
| **矿化固定** | 任意 | ⭐⭐⭐⭐ | 中 | 中 |

### 2.2 推荐技术：化学吸收法

**原理**：
```
吸收：R-NH₂ + CO₂ + H₂O → R-NH₃⁺ + HCO₃⁻
再生：R-NH₃⁺ + HCO₃⁻ + 热量 → R-NH₂ + CO₂↑ + H₂O
```

**工艺流程**：
```
垃圾焚烧烟气
    ↓
除尘 → 脱硝 → 脱酸 → 水洗 → 降温
    ↓
吸收塔（胺液吸收 CO2）
    ↓
净化气（CO2<5%）→ 烟囱排放
    ↓
富液（含 CO2）
    ↓
再生塔（加热解吸）
    ↓
贫液循环 ← ← ←
    ↓
高纯 CO2（99%+）→ 压缩 → 利用/封存
```

**关键参数**：

| 参数 | 数值 |
|------|------|
| CO2 回收率 | 85-95% |
| 产品纯度 | 98-99.9% |
| 蒸汽消耗 | 2.5-4.0 GJ/吨 CO2 |
| 电耗 | 80-150 kWh/吨 CO2 |
| 胺液损耗 | 0.5-1.5 kg/吨 CO2 |

**垃圾焚烧厂优势**：
- 有余热可用（降低再生能耗）
- 烟气处理系统完善
- 位于城市周边（靠近 CO2 用户）

### 2.3 技术挑战与解决

| 挑战 | 说明 | 解决方案 |
|------|------|---------|
| 烟气成分复杂 | 含 HCl、SO2、NOx 等 | 前端完善脱除工艺 |
| 粉尘影响 | 可能堵塞吸收塔 | 高效布袋除尘 |
| 二噁英风险 | 微量有机污染物 | 活性炭吸附 + 监测 |
| 胺液降解 | 杂质导致胺液氧化 | 添加抗氧化剂、定期补充 |
| 腐蚀问题 | 酸性气体腐蚀 | 选用耐蚀材料（316L） |

## 三、CO2 利用途径

### 3.1 利用途径总览

```
回收 CO2
   ↓
┌──┴──┬──────┬──────┬──────┬──────┬──────┐
↓     ↓      ↓      ↓      ↓      ↓      ↓
建材  食品   化工   驱油   微藻   尿素   甲醇
矿化  级     合成   (EOR) 养殖   合成   合成
```

### 3.2 建材矿化（最匹配）

**原理**：CO2 与碱性固废反应生成碳酸盐

**垃圾焚烧厂优势**：
- 自产飞灰/炉渣（碱性）
- 可协同处置其他工业固废
- 产品有市场（建材骨料）

**技术路线**：

#### （1）飞灰矿化
```
飞灰（含 CaO、MgO）+ CO2 → 碳酸钙 + 稳定化产物
```
| 指标 | 数值 |
|------|------|
| CO2 固定量 | 0.1-0.2 吨/吨飞灰 |
| 产品 | 碳酸钙、建材骨料 |
| 飞灰解毒 | 同时固化重金属 |
| 技术成熟度 | 示范推广 |

**优点**：
- 飞灰无害化（重金属固化）
- 固定 CO2
- 产品可用作建材

#### （2）钢渣/电石渣矿化
```
钢渣（含 CaO）+ CO2 → 碳酸钙 + 建材骨料
```
| 指标 | 数值 |
|------|------|
| CO2 固定量 | 0.15-0.25 吨/吨钢渣 |
| 产品 | 人工骨料、路基材料 |
| 市场规模 | 中等 |

#### （3）混凝土养护
```
预制混凝土 + CO2 → 碳酸化养护 → 强度提升
```
| 指标 | 数值 |
|------|------|
| CO2 用量 | 5-15 kg/m³ 混凝土 |
| 强度提升 | 15-25% |
| 养护时间 | 缩短 30-50% |
| 技术成熟度 | 商业化 |

**代表企业**：
- 加拿大 CarbonCure（全球 100+ 工厂）
- 中国宝武集团（钢渣矿化示范）

### 3.3 食品级 CO2

**纯度要求**：

| 指标 | 要求 |
|------|------|
| CO2 纯度 | ≥99.9% |
| CO | <5 ppm |
| H2S | <1 ppm |
| 总硫 | <3 ppm |
| 水分 | <50 ppm |
| 油分 | <0.5 ppm |
| 异味 | 无 |
| 有机物 | 极低 |

**垃圾焚烧厂挑战**：
- ⚠️ 烟气成分复杂，可能含二噁英等
- ⚠️ 公众接受度问题
- ⚠️ 需要深度纯化

**工艺流程**：
```
回收 CO2 → 脱硫 → 脱硝 → 脱卤 → 脱烃 → 脱水 → 精馏 → 液化
```

**投资估算**（1 万吨/年）：
| 项目 | 投资（万元） |
|------|-------------|
| 提纯系统 | 1000-1500 |
| 液化系统 | 400-600 |
| 灌装线 | 200-400 |
| 合计 | 1600-2500 |

**经济效益**：
| 项目 | 数值 |
|------|------|
| 售价 | 800-1500 元/吨 |
| 成本 | 500-700 元/吨 |
| 毛利 | 300-800 元/吨 |

**建议**：垃圾焚烧厂 CO2 用于食品级需谨慎，建议优先用于工业用途

### 3.4 化工原料

#### （1）尿素合成
```
CO2 + 2NH3 → CO(NH2)2 + H2O
```
| 指标 | 数值 |
|------|------|
| CO2 消耗 | 0.75 吨/吨尿素 |
| 技术 | 成熟 |
| 市场 | 大（肥料） |
| 要求 | 需配套合成氨 |

**适用场景**：垃圾焚烧厂附近有尿素厂

#### （2）甲醇合成
```
CO2 + 3H2 → CH3OH + H2O
```
| 指标 | 数值 |
|------|------|
| CO2 消耗 | 1.375 吨/吨甲醇 |
| H2 消耗 | 0.188 吨/吨甲醇 |
| 技术 | 示范推广 |
| 挑战 | 需要廉价绿氢 |

**适用场景**：有绿电/绿氢来源

#### （3）碳酸酯合成
```
CO2 + 环氧乙烷 → 碳酸乙烯酯
CO2 + 环氧丙烷 → 碳酸丙烯酯
```
| 指标 | 数值 |
|------|------|
| 产品价值 | 8000-15000 元/吨 |
| 技术 | 成熟 |
| 市场 | 中等（电池溶剂、塑料） |

#### （4）可降解塑料（PPC）
```
CO2 + 环氧丙烷 → 聚碳酸亚丙酯（PPC）
```
| 指标 | 数值 |
|------|------|
| CO2 含量 | 30-40%（质量比） |
| 产品价值 | 15000-25000 元/吨 |
| 技术 | 示范推广 |
| 应用 | 包装材料、农膜 |

### 3.5 微藻养殖

**原理**：微藻光合作用固定 CO2

| 指标 | 数值 |
|------|------|
| CO2 固定量 | 1.5-2.0 吨/吨藻粉 |
| 产品价值 | 藻粉 3000-8000 元/吨 |
| 技术 | 示范中 |
| 挑战 | 成本高、规模小 |

**垃圾焚烧厂优势**：
- 有余热（维持藻池温度）
- 有处理后的中水（藻类营养）
- 位于城市周边（靠近市场）

**产品**：
- 螺旋藻、小球藻（食品/饲料）
- 藻油（生物柴油）
- 虾青素、DHA（保健品）

### 3.6 驱油（EOR）

**原理**：CO2 注入油田提高原油采收率

| 指标 | 数值 |
|------|------|
| CO2 注入量 | 0.3-0.6 吨/吨原油增产 |
| 原油增产 | 10-20% |
| 封存率 | 50-70%（永久封存） |
| 经济半径 | <300 km |

**适用性**：垃圾焚烧厂通常位于城市周边，远离油田，适用性有限

### 3.7 温室种植

**原理**：CO2 气肥促进植物生长

| 指标 | 数值 |
|------|------|
| CO2 浓度 | 800-1200 ppm（大气 420 ppm） |
| 增产效果 | 20-40% |
| 用量 | 5-10 kg/亩/天 |
| 技术 | 成熟 |

**垃圾焚烧厂优势**：
- 位于城市周边（靠近温室基地）
- 有余热（冬季温室供暖）
- 可结合生态农业

| 来源 | 金额（万元/年） |
|------|---------------|
| CO2 销售 | 2000 |

**结论**：当前技术经济条件下，垃圾焚烧厂 CO2 回收利用**经济性较差**，需要：
1. 碳价大幅提升（>300 元/吨）
2. 技术突破降低能耗
3. 政策补贴支持
4. 高附加值利用途径

### 6.2 碳排放核算

**垃圾焚烧 CO2 核算规则**：

| 来源 | 核算方式 |
|------|---------|
| 生物质燃烧 | 不计入（碳中性） |
| 化石源（塑料） | 计入排放 |
| CO2 利用 | 可扣减排放量 |
| CO2 封存 | 可扣减排放量 |

**核算公式**：
```
净排放 = 化石源 CO2 排放 - CO2 利用/封存量
```

### 7.1 低能耗吸收剂

### 7.2 余热利用

**垃圾焚烧厂优势**：有余热可用于 CO2 再生

| 热源 | 温度 | 可用性 |
|------|------|--------|
| 蒸汽轮机抽汽 | 120-150°C | ⭐⭐⭐⭐⭐ |
| 烟气余热 | 150-200°C | ⭐⭐⭐⭐ |
| 热水 | 80-100°C | ⭐⭐⭐ |

**节能潜力**：利用余热可降低能耗成本 30-50%

### 7.3 一体化设计

**趋势**：新建垃圾焚烧厂预留 CO2 捕集接口

```
垃圾焚烧炉 → 余热锅炉 → 发电
              ↓
          蒸汽抽汽 → CO2 再生
              ↓
          烟气处理 → CO2 捕集
```

### 7.4 负排放技术（BECCS）

**原理**：垃圾中生物质燃烧产生 CO2 + 捕集封存 = 负排放

| 指标 | 数值 |

## 八、结论与建议

### 8.1 技术可行性

✅ **技术上可行**
- 化学吸收法成熟可靠
- 垃圾焚烧厂有余热可利用
- 可结合飞灰矿化协同处置

### 8.2 经济可行性

⚠️ **当前经济性较差，需要政策支持**

| 情景 | 经济性 |
|------|--------|
| 当前碳价（80 元/吨） | 亏损 600-700 元/吨 |
| 碳价 200 元/吨 | 亏损 300-400 元/吨 |
| 碳价 300 元/吨 +CCER | 基本盈亏平衡 |
| 碳价 500 元/吨 | 盈利 100-200 元/吨 |

### 8.3 推荐利用途径（优先级）

| 优先级 | 利用途径 | 理由 |
|--------|---------|------|
| 1️⃣ | 建材矿化（飞灰/钢渣） | 协同处置固废、技术可行 |
| 2️⃣ | 温室气肥 | 就近利用、技术成熟 |
| 3️⃣ | 混凝土养护 | 商业化程度高 |
| 4️⃣ | 工业级 CO2 销售 | 有市场时考虑 |
| 5️⃣ | 食品级 CO2 | 谨慎（公众接受度） |

### 8.4 政策建议

1. **碳市场纳入**：将垃圾焚烧纳入全国碳市场
2. **CCER 支持**：简化 CO2 利用项目 CCER 开发流程
3. **财政补贴**：对示范项目给予投资补贴
4. **标准制定**：完善 CO2 产品质量标准
5. **技术研发**：支持低能耗捕集技术研发

### 8.5 企业行动建议

| 企业类型 | 建议 |
|---------|------|
| 新建垃圾焚烧厂 | 预留 CO2 捕集接口和场地 |
| 现有大型厂 | 开展小规模示范（1-5 万吨/年） |
| 城市周边厂 | 优先开发温室气肥、建材利用 |
