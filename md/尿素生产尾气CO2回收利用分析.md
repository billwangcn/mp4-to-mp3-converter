# 尿素生产尾气 CO2 回收利用分析

**整理时间**: 2026-03-29  
**信息来源**: 公开技术资料、行业报告、学术论文

## 一、尿素生产尾气来源与成分

### 1.1 尾气产生环节

尿素生产过程中，CO2 主要在以下环节产生或排放：

```
合成氨厂 → CO₂ 脱除 → 尿素合成 → 分解回收 → 解析尾气
              ↓            ↓           ↓
          脱碳尾气    未反应 CO₂   低压放空
```

| 环节 | 来源 | CO2 含量 | 排放量 |
|------|------|---------|--------|
| **合成氨脱碳** | 原料气净化 | 95-99% | 主要来源 |
| **尿素合成** | 未反应 CO2 循环 | 内部循环 | - |
| **高压分解** | 未反应物分解 | 60-80% | 回收循环 |
| **中低压分解** | 深度分解 | 30-50% | 部分回收 |
| **解析尾气** | 最终排放 | 15-30% | 主要排放点 |
| **造粒粉尘** | 尿素分解 | 微量 | 少量 |

### 1.2 各排放点 CO2 浓度

#### （1）合成氨脱碳尾气（最大来源）

合成氨生产中，原料气（煤/天然气）经变换后产生大量 CO2，需脱除才能合成氨。

| 脱碳工艺 | CO2 浓度 | 压力 | 温度 | 纯度 |
|---------|---------|------|------|------|
| 物理吸收法（低温甲醇） | 95-99% | 常压 | 常温 | 高 |
| 化学吸收法（MDEA） | 95-98% | 常压 | 40-60°C | 高 |
| 变压吸附（PSA） | 90-95% | 常压 | 常温 | 中 |
| 膜分离法 | 80-90% | 常压 | 常温 | 低 |

**排放量**：生产 1 吨合成氨约副产 1.8-2.2 吨 CO2

#### （2）尿素装置解析尾气

尿素生产中未反应的 NH3 和 CO2 经三级分解回收后，残余气体从解析塔顶排出。

| 组分 | 体积浓度 | 说明 |
|------|---------|------|
| CO2 | 15-30% | 主要成分 |
| NH3 | 0.5-2% | 需回收 |
| H2O | 50-70% | 水蒸气 |
| N2+O2 | 5-10% | 空气漏入 |
| 惰性气体 | 5-10% | Ar、CH4 等 |

**排放量**：生产 1 吨尿素约排放 50-100 Nm³ 尾气

#### （3）其他排放点

| 排放点 | CO2 浓度 | 特点 |
|--------|---------|------|
| 蒸发冷凝液 | 溶解 CO2 | 浓度低，难回收 |
| 真空系统 | 10-20% | 间歇排放 |
| 设备泄漏 | - | 无组织排放 |

### 1.3 CO2 平衡图（以 1 吨尿素计）

```
                    合成氨来 CO₂: 750-800 kg
                           ↓
              ┌────────────────────────┐
              │      尿素合成塔        │
              │   CO₂ 转化率 58-65%    │
              └───────────┬────────────┘
                          ↓
          ┌───────────────┼───────────────┐
          ↓               ↓               ↓
    反应生成尿素    未反应 CO₂       副产/损失
    (消耗 CO₂)      (回收循环)       (排放)
    733 kg          ~60-70%         ~5-10%
                                    40-80 kg
```

**总排放**：生产 1 吨尿素，全生命周期排放 CO2 约：
- 煤制路线：2.5-3.0 吨 CO2
- 天然气路线：1.2-1.5 吨 CO2
- 绿氢路线：0.1-0.3 吨 CO2

## 二、CO2 回收技术

### 2.1 回收技术路线对比

| 技术 | 原理 | 适用浓度 | 能耗 | 投资 | 成熟度 |
|------|------|---------|------|------|--------|
| **化学吸收法** | 胺液吸收 CO2 | 5-30% | 高 | 中 | 成熟 |
| **物理吸收法** | 溶剂物理溶解 | >30% | 中 | 中 | 成熟 |
| **变压吸附** | 分子筛选择性吸附 | 10-50% | 中 | 低 | 较成熟 |
| **膜分离法** | 膜选择性透过 | >20% | 低 | 低 | 发展中 |
| **低温蒸馏** | 低温液化分离 | >50% | 高 | 高 | 成熟 |
| **矿化固定** | 化学反应固定 | 任意 | 中 | 中 | 示范 |

### 2.2 化学吸收法（最成熟）

**原理**：
```
吸收：R-NH₂ + CO₂ + H₂O → R-NH₃⁺ + HCO₃⁻
再生：R-NH₃⁺ + HCO₃⁻ + 热量 → R-NH₂ + CO₂↑ + H₂O
```

**常用吸收剂**：

| 吸收剂 | 浓度 | 吸收容量 | 再生能耗 | 特点 |
|--------|------|---------|---------|------|
| MEA（一乙醇胺） | 15-30% | 高 | 高 | 技术最成熟 |
| MDEA（甲基二乙醇胺） | 30-50% | 中 | 中 | 选择性好 |
| 活化 MDEA | 30-45% | 高 | 中 | 改进型 |
| 复合胺 | 定制 | 高 | 中低 | 最新技术 |

**工艺流程**：
```
尾气 → 除尘 → 吸收塔（胺液吸收 CO2） → 净化气排放
                    ↓
              富液（含 CO2）
                    ↓
              再生塔（加热解吸）
                    ↓
              贫液循环 ← ← ← ← ←
                    ↓
              高纯 CO2（99%+）
```

**关键参数**：
| 参数 | 数值 |
|------|------|
| CO2 回收率 | 85-95% |
| 产品纯度 | 98-99.9% |
| 蒸汽消耗 | 2.5-4.0 GJ/吨 CO2 |
| 电耗 | 80-150 kWh/吨 CO2 |
| 胺液损耗 | 0.5-1.5 kg/吨 CO2 |

**优点**：
- 技术成熟，应用广泛
- 可处理低浓度 CO2（5% 以上）
- 产品纯度高

**缺点**：
- 能耗高（主要是再生蒸汽）
- 胺液降解、腐蚀问题
- 设备体积大

### 2.3 变压吸附法（PSA）

**原理**：利用分子筛在不同压力下对 CO2 的吸附容量差异

**工艺流程**：
```
尾气 → 压缩 → 吸附塔（CO2 被吸附） → 净化气
                    ↓
              降压解吸
                    ↓
              CO2 产品气
```

**关键参数**：
| 参数 | 数值 |
|------|------|
| CO2 回收率 | 75-90% |
| 产品纯度 | 90-98% |
| 电耗 | 150-250 kWh/吨 CO2 |
| 吸附剂寿命 | 3-5 年 |

**优点**：
- 能耗相对较低（无蒸汽消耗）
- 流程简单，自动化程度高
- 无化学药剂消耗

**缺点**：
- 适合中高浓度 CO2（>10%）
- 产品纯度相对较低
- 吸附剂需定期更换

### 2.4 膜分离法

**原理**：利用高分子膜对 CO2 的选择性透过

**膜材料**：
| 材料 | CO2/N2 选择性 | 渗透通量 | 耐温 |
|------|-------------|---------|------|
| 聚酰亚胺 | 20-40 | 中 | 好 |
| 聚砜 | 15-30 | 高 | 中 |
| 纤维素醋酸 | 25-50 | 中 | 差 |
| 混合基质膜 | 40-100 | 中 | 好 |

**工艺流程**：
```
尾气 → 压缩 → 膜组件 → 渗透气（富 CO2）
                    ↓
              渗余气（贫 CO2）排放
```

**关键参数**：
| 参数 | 数值 |
|------|------|
| CO2 回收率 | 60-85% |
| 产品纯度 | 70-95% |
| 电耗 | 100-200 kWh/吨 CO2 |
| 膜寿命 | 3-5 年 |

**优点**：
- 能耗最低
- 设备紧凑，无运动部件
- 操作简单

**缺点**：
- 适合中高浓度（>20%）
- 产品纯度受限
- 膜污染问题

### 2.5 技术选择建议

| 尾气来源 | CO2 浓度 | 推荐技术 | 理由 |
|---------|---------|---------|------|
| 合成氨脱碳气 | 95-99% | 直接压缩 | 纯度高，无需分离 |
| 尿素解析气 | 15-30% | 化学吸收法 | 浓度低，吸收法最有效 |
| 混合尾气 | 20-50% | PSA 或膜分离 | 能耗较低 |
| 小规模（<1 万吨/年） | 任意 | PSA | 投资低，操作简单 |
| 大规模（>10 万吨/年） | 任意 | 化学吸收 | 规模效应，成本低 |

## 三、CO2 利用途径

### 3.1 主要利用方向

```
回收 CO2
   ↓
┌──┴──┬──────┬──────┬──────┬──────┐
↓     ↓      ↓      ↓      ↓      ↓
尿素  食品   化工   驱油   微藻   建材
合成  级     合成   (EOR) 养殖   矿化
```

### 3.2 回注尿素生产（最经济）

**方案**：将回收的 CO2 补入尿素合成系统

| 指标 | 数值 |
|------|------|
| CO2 需求 | 0.75 吨/吨尿素 |
| 回收成本 | 150-300 元/吨 CO2 |
| 尿素增值 | 约 500-800 元/吨 CO2 |
| 投资回收期 | 2-4 年 |

**优点**：
- 无需外销市场
- 工艺成熟
- 经济效益好

**限制**：
- 只能回收装置自身排放
- 受尿素产能限制

### 3.3 食品级 CO2

**纯度要求**：
| 指标 | 要求 |
|------|------|
| CO2 纯度 | ≥99.9% |
| CO | <5 ppm |
| H2S | <1 ppm |
| 总硫 | <3 ppm |
| 水分 | <50 ppm |
| 油分 | <0.5 ppm |
| 异味 | 无 |

**工艺流程**：
```
回收 CO2 → 脱硫 → 脱烃 → 脱水 → 精馏 → 液化 → 灌装
```

**投资估算**（1 万吨/年）：
| 项目 | 投资（万元） |
|------|-------------|
| 提纯系统 | 800-1200 |
| 液化系统 | 300-500 |
| 灌装线 | 200-400 |
| 合计 | 1300-2100 |

**经济效益**：
| 项目 | 数值 |
|------|------|
| 售价 | 800-1500 元/吨 |
| 成本 | 400-600 元/吨 |
| 毛利 | 400-900 元/吨 |
| 投资回收期 | 3-5 年 |

**市场**：饮料、食品保鲜、烟草膨胀等

### 3.4 化工原料

#### （1）甲醇合成
```
CO2 + 3H2 → CH3OH + H2O
```
| 指标 | 数值 |
|------|------|
| CO2 消耗 | 1.375 吨/吨甲醇 |
| H2 消耗 | 0.188 吨/吨甲醇 |
| 技术 | 成熟（已有工业装置） |
| 挑战 | 需要廉价绿氢 |

#### （2）碳酸酯合成
```
CO2 + 环氧乙烷 → 碳酸乙烯酯
CO2 + 环氧丙烷 → 碳酸丙烯酯
```
| 指标 | 数值 |
|------|------|
| 产品价值 | 高（8000-15000 元/吨） |
| 市场规模 | 中等 |
| 技术 | 成熟 |

#### （3）可降解塑料（PPC）
```
CO2 + 环氧丙烷 → 聚碳酸亚丙酯（PPC）
```
| 指标 | 数值 |
|------|------|
| CO2 含量 | 30-40%（质量比） |
| 产品价值 | 15000-25000 元/吨 |
| 技术 | 示范推广中 |
| 市场 | 包装材料、农膜 |

### 3.5 驱油（EOR）

**原理**：将 CO2 注入油田，提高原油采收率

| 指标 | 数值 |
|------|------|
| CO2 注入量 | 0.3-0.6 吨/吨原油增产 |
| 原油增产 | 10-20% |
| 封存率 | 50-70%（永久封存） |
| 经济半径 | <300 km（管道输送） |

**国内主要油田**：
| 油田 | 位置 | CO2 需求 |
|------|------|---------|
| 大庆油田 | 黑龙江 | 大 |
| 胜利油田 | 山东 | 大 |
| 吉林油田 | 吉林 | 中 |
| 长庆油田 | 陕西 | 中 |
| 新疆油田 | 新疆 | 大 |

**经济效益**：
| 项目 | 数值 |
|------|------|
| CO2 售价 | 200-400 元/吨 |
| 管道输送成本 | 50-150 元/吨 |
| 净收益 | 50-250 元/吨 |

### 3.6 微藻养殖

**原理**：微藻光合作用固定 CO2，生产生物质

| 指标 | 数值 |
|------|------|
| CO2 固定量 | 1.5-2.0 吨/吨藻粉 |
| 产品价值 | 藻粉 3000-8000 元/吨 |
| 技术 | 示范中 |
| 挑战 | 成本高、规模小 |

**产品**：
- 螺旋藻、小球藻（食品/饲料）
- 藻油（生物柴油）
- 虾青素、DHA（保健品）

### 3.7 建材矿化

**原理**：CO2 与碱性物质反应生成碳酸盐

| 技术路线 | 产品 | 成熟度 |
|---------|------|--------|
| 钢渣矿化 | 碳酸钙、建材骨料 | 示范 |
| 电石渣矿化 | 碳酸钙 | 示范 |
| 混凝土养护 | 增强混凝土 | 商业化 |
| 人工骨料 | 轻质骨料 | 研发 |

**优点**：
- 永久封存 CO2
- 消纳工业固废
- 产品有市场

**缺点**：
- 反应速率慢
- 能耗较高
- 市场规模有限

## 七、结论与建议

### 7.1 技术可行性

✅ **CO2 回收技术成熟可靠**
- 化学吸收法：适用低浓度，技术最成熟
- PSA/膜分离：适用中高浓度，能耗较低
- 食品级提纯：可满足最高标准要求

### 7.2 经济可行性

⚠️ **取决于利用途径和规模**

| 利用途径 | 经济性 | 建议 |
|---------|--------|------|
| 回注尿素 | ⭐⭐⭐⭐⭐ | 优先推荐 |
| 食品级 | ⭐⭐⭐⭐ | 有市场时推荐 |
| 驱油 | ⭐⭐⭐ | 靠近油田时考虑 |
| 甲醇合成 | ⭐⭐ | 需廉价绿氢 |
| 化工合成 | ⭐⭐⭐ | 高附加值产品可行 |
| 微藻/建材 | ⭐⭐ | 示范阶段 |

### 7.3 推荐方案

**尿素企业 CO2 回收优先级**：

1. **第一优先**：回注尿素生产
   - 技术成熟、收益稳定
   - 规模匹配、无需外销

2. **第二优先**：食品级 CO2
   - 附加值高
   - 需开发销售渠道

3. **第三优先**：驱油（EOR）
   - 靠近油田时考虑
   - 需建设管道

4. **长期布局**：绿氢耦合制甲醇
   - 等待绿氢成本下降
   - 提前规划技术路线

### 7.4 关键成功因素

| 因素 | 要求 |
|------|------|
| 尾气 CO2 浓度 | >15%（化学吸收）、>30%（PSA/膜） |
| 规模 | >5 万吨/年（有规模效应） |
| 利用市场 | 稳定销售渠道 |
| 能源成本 | 蒸汽<200 元/吨、电<0.6 元/kWh |
| 碳价 | >100 元/吨时有额外收益 |
